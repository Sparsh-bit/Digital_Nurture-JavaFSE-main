# JUnit Basic Testing Exercise

This project demonstrates the basic setup of a Java project using Maven and JUnit 4, specifically designed as a foundational exercise for testing in Java.

## Project Structure
- **Build Tool:** Maven (`pom.xml`)
- **Testing Framework:** JUnit 4.13.2
- **Source Code:** `src/main/java`
- **Test Code:** `src/test/java`

## Getting Started

1. **Clone the repository.**
2. **Open the project** in your preferred IDE (e.g., IntelliJ IDEA, Eclipse).
3. **Resolve Maven Dependencies:** The IDE will automatically download the JUnit dependency defined in the `pom.xml`.
4. **Run Tests:** Right-click on `CalculatorTest.java` and select "Run 'CalculatorTest'" to execute the unit tests.

## Purpose
Demonstrates how to:
1. Create a Java Maven project.
2. Add the JUnit dependency.
3. Write standard unit tests using the `@Test` annotation and assertions.
