# Exercise 12: Spacing Utilities

## Objective
Exercise 12.1: Apply margin (`m-3`, `mt-4`) and padding (`p-2`, `py-5`) utilities on layout sections.
Exercise 12.2: Build a pricing section where spacing improves the layout.

## Output & Explanation
- **Output**: Examples of simple boxes with varying margins and padding. Followed by a polished pricing table layout that heavily utilizes spacing classes to give elements room to breathe.
- **Explanation**:
  - Spacing utilities use a format of `{property}{sides}-{size}` (e.g., `m` for margin, `t` for top, `4` for size).
  - In the pricing section, classes like `p-4` (padding inside card), `mt-3` (margin-top on price), and `mb-4` (margin-bottom to separate rows/elements) are used to construct a professional, well-spaced design without custom CSS.
