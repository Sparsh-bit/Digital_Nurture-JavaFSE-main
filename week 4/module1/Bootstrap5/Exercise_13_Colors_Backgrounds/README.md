# Exercise 13: Colors and Backgrounds

## Objective
Exercise 13.1: Create a dashboard page using contextual background classes (`bg-primary`, `bg-warning`, etc.) and text colors.
Exercise 13.2: Use `bg-gradient` with `bg-dark` and white text.

## Output & Explanation
- **Output**: A mini dashboard layout featuring four metric cards with distinct contextual background colors, and a large hero section with a dark gradient.
- **Explanation**:
  - `bg-*` classes apply background colors mapped to Bootstrap's theme palette.
  - Note how `bg-warning` pairs with `text-dark` because warning yellow has poor contrast with white text.
  - `bg-gradient` adds a linear gradient to the background, creating a more modern 3D appearance.
