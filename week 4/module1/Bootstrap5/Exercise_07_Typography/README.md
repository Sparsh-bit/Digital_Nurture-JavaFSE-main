# Exercise 7: Typography

## Objective
Exercise 7.1: Create a sample page with different Bootstrap typography utilities: `display-1`, `lead`, `text-muted`, `fw-bold`, etc.
Exercise 7.2: Use `text-uppercase`, `text-lowercase`, `text-capitalize`.

## Output & Explanation
- **Output**: A page showcasing huge display headings, emphasized lead paragraphs, muted text, and various text case transformations.
- **Explanation**:
  - `.display-1` is used for massive, standout headings.
  - `.lead` makes a paragraph slightly larger and thinner.
  - `.text-muted` changes the color to a lighter gray.
  - `.fw-bold` applies `font-weight: bold`.
  - The text transformation classes automatically convert string casing using CSS `text-transform`.
