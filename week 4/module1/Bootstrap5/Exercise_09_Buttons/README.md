# Exercise 9: Buttons

## Objective
Exercise 9.1: Create buttons using all contextual classes: `btn-primary`, `btn-secondary`, `btn-outline-*`, etc.
Exercise 9.2: Add button groups using `btn-group`, and create toggle buttons with checkboxes.

## Output & Explanation
- **Output**: A showcase of all colored buttons (solid and outline variants), followed by a joined button group, and a toggleable button group functioning as checkboxes.
- **Explanation**:
  - `btn` provides the base button styling, while `.btn-*` provides the specific semantic color. `.btn-outline-*` makes the background transparent until hovered.
  - `.btn-group` wraps multiple buttons and removes border-radius between them to visually join them.
  - Combining `btn-check` on an input and `btn` on a label allows us to create stylized toggle buttons that act like checkboxes under the hood.
