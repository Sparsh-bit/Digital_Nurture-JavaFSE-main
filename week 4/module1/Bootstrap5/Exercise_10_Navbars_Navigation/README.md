# Exercise 10: Navbars and Navigation

## Objective
Exercise 10.1: Create a responsive navbar with logo, navigation links, and a search form.
Exercise 10.2: Use `nav`, `nav-tabs`, `nav-pills` for creating tabbed navigation.

## Output & Explanation
- **Output**: 
  - A top dark navbar that collapses into a hamburger menu on smaller screens, containing a brand, links, and a search form.
  - Demonstrations of tab and pill navigations.
- **Explanation**:
  - `navbar-expand-lg` dictates that the navbar will remain expanded on large screens and collapse on medium/small screens.
  - `.nav-tabs` creates a bordered tab interface, while `.nav-pills` creates background-colored rounded buttons for active states.
