# Exercise 8: Event Handling

This exercise connects standard user inputs to JavaScript events.

I used `.addEventListener()` for three different scenarios:
1. `keydown`: On the search text box, I capture every keystroke and immediately update the output text to simulate a live search.
2. `change`: On the dropdown menu, the event fires whenever the user selects a new category, which could be used to filter a list.
3. `click`: The simplest one—when the register button is clicked, it just updates the output text.
