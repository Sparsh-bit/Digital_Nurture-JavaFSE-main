# Exercise 6: Arrays and Methods

I practiced some standard array manipulation here. 

I started with a basic array of objects. Then:
- Used `.push()` to inject a new "Piano Recital" event into the array.
- Used `.filter()` with a tiny arrow function to create a new array containing ONLY events where the category is "Music".
- Used `.map()` to loop through the entire array and transform each object into a formatted string (like "Event Card: [Food] Baking Basics"). `.map()` is super useful for converting raw data into UI text.
