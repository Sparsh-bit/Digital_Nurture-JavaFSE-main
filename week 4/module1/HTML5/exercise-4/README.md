# Exercise 4: Image Gallery

I built a simple 2x3 image gallery to show off past community events.

Instead of using flexbox or CSS grid, I went old-school and used an HTML `<table>` as requested. The table has 2 rows (`<tr>`) and 3 columns (`<td>`) per row.

For the images, I made sure to include the `alt` text (for accessibility) and `title` attributes (so a tooltip appears when you hover over them). I styled all the images using a `.gallery-img` class to give them a dark border and rounded corners. Lastly, I threw in a `.caption` div under each image to describe the event.
