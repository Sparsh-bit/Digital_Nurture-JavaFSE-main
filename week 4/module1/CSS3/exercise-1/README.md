# Exercise 1: CSS Inclusion Methods

Here I tested out the three different ways to add CSS to our Community Event Portal.

- First, I used an **inline style** directly on the `<h1>` tag to make the text red (`style="color: red;"`).
- Next, I used an **internal stylesheet** (`<style>`) in the `<head>` to set a light background color on the body.
- Finally, I linked an **external stylesheet** (`styles.css`) where I moved the reusable stuff, like the button styling and basic header padding, and labeled them with comments.
