# Exercise 38: Decompile a Class File

**Instructions:**
1. Compile the file: `javac SecretLogic.java`
2. Download a decompiler like JD-GUI or CFR.
3. Open `SecretLogic.class` in the tool.
You'll see that the hardcoded password "SuperSecret123" is plainly visible in the decompiled source! This is why you never hardcode secrets in Java!
