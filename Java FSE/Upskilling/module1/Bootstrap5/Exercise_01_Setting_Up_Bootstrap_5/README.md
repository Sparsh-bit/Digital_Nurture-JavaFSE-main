# Exercise 1: Setting Up Bootstrap 5

## Objective
Exercise 1.1: Create a basic HTML page and link Bootstrap 5 via CDN.
Exercise 1.2: Set up a project using npm or downloaded Bootstrap files. Use the downloaded files in a sample HTML page.

## Output & Explanation
- **Output**: A simple webpage displaying a welcome heading and a paragraph styled with Bootstrap's default typography and a container class.
- **Explanation**: 
  - `index.html` implements Exercise 1.1 by linking Bootstrap 5 CSS and JS (bundle) via jsdelivr CDN.
  - For Exercise 1.2, you would typically run `npm install bootstrap` in a project directory, which would create a `node_modules` folder containing Bootstrap's CSS and JS files. You would then link these local files in your HTML. We are simulating the CDN approach here for immediate visual results.
