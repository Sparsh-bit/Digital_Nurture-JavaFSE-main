# Exercise 15: Borders, Shadows, and Rounded Corners

## Objective
Exercise 15.1: Add border utilities like `border`, `border-primary`, `border-3`, and `rounded-circle` to an image.
Exercise 15.2: Use `shadow`, `shadow-lg` and `rounded-pill` in a card.

## Output & Explanation
- **Output**: An image masked as a perfect circle with a thick primary-colored border. Below it is an elevated card with a large shadow, containing a pill-shaped button.
- **Explanation**:
  - `border` activates the border, `border-primary` colors it blue, and `border-3` thickens it.
  - `rounded-circle` turns a square element into a circle (50% border-radius).
  - `shadow-lg` applies a large box-shadow.
  - `rounded-pill` makes elements like buttons pill-shaped by applying a massive border radius.
