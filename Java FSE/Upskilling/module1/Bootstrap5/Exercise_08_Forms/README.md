# Exercise 8: Forms

## Objective
Exercise 8.1: Create a registration form using Bootstrap form components like `form-control`, `form-check`, and `input-group`.
Exercise 8.2: Style a login form using `form-floating`.

## Output & Explanation
- **Output**: A page split into two sections: a registration form with standard inputs and an input group (@username), and a login form with modern floating labels.
- **Explanation**:
  - `form-control` applies styling to textual inputs like width, border, padding, and focus states.
  - `input-group` allows appending or prepending text (like `@`) to an input.
  - `form-check` is specifically for styling checkboxes and radios properly.
  - `form-floating` allows labels to float inside the input field and shrink when the input is focused or filled, creating a sleek UI without requiring JavaScript.
