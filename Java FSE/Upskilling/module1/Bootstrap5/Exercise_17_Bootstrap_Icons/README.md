# Exercise 17: Icons with Bootstrap Icons

## Objective
Exercise 17.1: Install and use Bootstrap Icons in a webpage: add social media icons in the footer.
Exercise 17.2: Replace text buttons with icon-only buttons using Bootstrap Icons.

## Output & Explanation
- **Output**: 
  - A set of three action buttons (Edit, Delete, Settings) that contain only icons instead of text.
  - A dark footer featuring four social media icons (Twitter, Facebook, Instagram, LinkedIn).
- **Explanation**:
  - We included the Bootstrap Icons CSS via CDN in the `<head>`.
  - Icons are added using the `<i>` tag with classes like `bi bi-twitter`. They act like text, meaning we can style their size and color using standard Bootstrap text utilities (`fs-4`, `text-white`).
