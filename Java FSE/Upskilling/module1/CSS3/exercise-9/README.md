# Exercise 9: Multiple Columns in Text

The community wanted a bulletin board that reads like a traditional newspaper. 

Instead of dealing with flexbox or floats for columns of text, I used the CSS3 column properties. By applying `column-count: 2`, `column-gap: 40px`, and `column-rule: 1px solid gray` to the `.news-article` container, the browser automatically flows the paragraphs between two columns and draws a nice divider line down the middle. It's super easy and responsive!
