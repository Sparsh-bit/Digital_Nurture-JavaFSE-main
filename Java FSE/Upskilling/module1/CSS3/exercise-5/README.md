# Exercise 5: Typography

The marketing team wanted better fonts, so I brought in Google Fonts using `@import` at the top of the CSS file.

I assigned the 'Montserrat' font to the headers and made it bold, uppercase, and added some letter-spacing to make it look like a real headline. 

For the description text, I used 'Open Sans', bumped up the line-height for readability, and added a slight italic style. `text-align: justify` makes the paragraph block look really neat and newspaper-like.
