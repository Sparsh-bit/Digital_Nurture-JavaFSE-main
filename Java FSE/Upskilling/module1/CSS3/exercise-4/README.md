# Exercise 4: Color & Background Styling

I updated the theme to match the city council's branding vibes.

I added a subtle pattern background to the `body` and included a fallback hex color just in case the image doesn't load. For the main header, I applied a CSS linear gradient (`linear-gradient(...)`) which looks a lot more modern than a flat color.

I also mixed in HEX codes (`#ffffff`) for solid text colors and RGBA (`rgba(255, 255, 255, 0.9)`) to create a semi-transparent white background box so the pattern slightly bleeds through behind the content.
