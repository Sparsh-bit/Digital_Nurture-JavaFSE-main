# Exercise 1: HTML5 Base Template

Here is the base template I set up for the Local Community Event Portal. 

I made sure to include the proper HTML5 doctype (`<!DOCTYPE html>`), set the language to English, and included the UTF-8 charset meta tag so it renders characters correctly across all browsers.

I also added standard semantic HTML5 tags like `<nav>`, `<main>`, and `<footer>`, and put comments above them as requested so the team knows exactly where everything goes. If you open `index.html` in Chrome and hit F12 to inspect the document structure, you'll see the tree is super clean and semantic.
