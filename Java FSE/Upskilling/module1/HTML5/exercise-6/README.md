# Exercise 6: Event Handling

This exercise was all about wiring up JavaScript events to HTML elements. I built a mock feedback form to demonstrate them all:

1. **`onblur`**: When you click out of the Phone Number field, it triggers a validation script. If you typed less than 10 characters, it shows an error message.
2. **`onchange`**: When you select a different event from the dropdown, it instantly updates the text below it to show the correct fee.
3. **`onkeyup`**: As you type in the feedback textarea, a script counts the characters and updates the counter in real-time.
4. **`onclick`**: Clicking the submit button triggers a confirmation message.
5. **`ondblclick`**: If you double-click the promo image at the bottom, it toggles a CSS class that scales the image up by 1.5x. Double click it again to shrink it back!
