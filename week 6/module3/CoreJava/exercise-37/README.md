# Exercise 37: Using javap to Inspect Bytecode

**Instructions:**
1. Compile the file: `javac BytecodeDemo.java`
2. Run javap: `javap -c BytecodeDemo.class`

You'll see assembly-like instructions such as `iload_1`, `iload_2`, `iadd`, `ireturn` which represent loading ints onto the stack, adding them, and returning the result.
